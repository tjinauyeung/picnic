#### Dependencies

- [Node.js](https://nodejs.org/en/)
- [NVM](https://github.com/nvm-sh/nvm) (optional)

#### Get started

Use node version in `.nvmrc`

```
nvm use
```

Install dependencies
```
npm install
```

Start app
```
npm run start
```

Run tests
```
npm run test
```
